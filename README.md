# Spider Clean

Lightweight **Mac system cleaner** for the terminal.

```text
spider-clean clean
```

## Install

```bash
brew install anshpatelkb/spider-clean/spider-clean
```

One command. No custom tap URL, no `--HEAD`.

See [BREW.md](BREW.md) if brew asks you to trust the tap.

## Usage

```bash
spider-clean                 # menu
spider-clean clean           # cleanup + notification
spider-clean clean --dry-run
spider-clean status
spider-clean --help
```

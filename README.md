# Spider Clean

Lightweight **Mac system cleaner** for the terminal.

```text
spider-clean clean
```

## Install

```bash
brew install anshpatelkb/spider-clean/spider-clean
```

One command (repo is `homebrew-spider-clean`). No tap URL, no `--HEAD`.

If brew asks to trust the tap: `brew trust anshpatelkb/spider-clean` then install again.

## Usage

```bash
spider-clean                 # menu
spider-clean clean           # cleanup + notification
spider-clean clean --dry-run
spider-clean status
spider-clean --help
```

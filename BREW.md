# Install (one command)

```bash
brew install anshpatelkb/spider-clean/spider-clean
```

- No custom `brew tap … https://…`
- No `--HEAD`
- Version **1.3.1** from a fixed GitHub commit archive

If brew asks to trust the tap once:

```bash
brew trust anshpatelkb/spider-clean
brew install anshpatelkb/spider-clean/spider-clean
```

```bash
spider-clean --version
spider-clean clean
```
